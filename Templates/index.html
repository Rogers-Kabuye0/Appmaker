Forms

Material design forms are known for the animations. Google textinputs almost all animate, this is included in our kit. But you need to add some CSS to fit you app look.
Remember you are a programmer.

You need to add some consecutive classes to archive this and in time to come you will have to temper with the js file to add yours by just following the documentation.
Our js file is well formated and readable since we use alot of jquery.

First the outline has been removed from the bootstrap form.

	         						
	         						
<form class="form-group">
	<input type="text" class="form-control" name="">
</form>

Inputs with floating labels.
Accually, placeholders may not be needed and you can apply a feedback from the some label below.

	         						
	         						
<form class="form-group">
	<div class="form-floated">
		<input type="email" class="form-control">
		<label class="label-floated" for="floatingInput">Email address</label>
		<label for="floatingInput" class="label-base">Feedback</label>
	</div>

	<div class="form-floated mt-5">
		<input type="email" class="form-control">
		<label class="label-floated" for="floatingInput">Password</label>
		<label for="floatingInput" class="label-base text-success">feedback</label>
	</div>
</form>
